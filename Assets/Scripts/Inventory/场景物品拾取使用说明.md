# 场景物品拾取（WorldItemPickup）使用说明

## 脚本做什么

- 玩家**进入**拾取物的碰撞范围时，把 `ItemData` 用 `Inventory.AddItem` 加入背包。
- 默认可选 **走进即捡** 或 **按 E 再捡**。
- 全部装入背包后**销毁**地上的物体；装不下会保留在原地（或见下方选项）。

---

## 一、前置条件（项目里应已满足）

1. **Player** 的 Tag 为 **`Player`**（与 `EnemyBase` 里查找玩家一致）。
2. **Player**（或其父物体）上挂了 **`Inventory`** 组件。
3. **Player** 上有 **Collider2D**（不必是 Trigger），用于和拾取物重叠。
4. 已创建至少一个 **Item Data**（且 **id** 已填写）。

---

## 二、创建拾取物 Prefab

1. **Hierarchy** 右键 → **2D Object** → **Sprite**（或空物体 + **Sprite Renderer**）。
2. 命名，例如 `Pickup_Bone`。
3. **Add Component** → **Box Collider 2D** 或 **Circle Collider 2D**：
   - 勾选 **Is Trigger**。
   - 调整大小盖住精灵。
4. **Add Component** → **`World Item Pickup`**。
5. 在 **Item** 槽拖入你的 **Item Data** 资源。
6. **Count** 填数量（默认 1）。
7. **Require Interact Key**：
   - **不勾选**：走进范围立刻拾取。
   - **勾选**：走进范围后按 **E**（可在脚本里改 `Interact Key`）拾取。
8. 将物体从 **Hierarchy** 拖到 **Project** 做成 **Prefab**（建议放在 `Assets/Prefabs/Pickups/` 等文件夹）。

---

## 三、放进场景测试

1. 把 Prefab **拖进场景**任意位置。
2. **Play**。
3. 用玩家走过去：
   - 自动拾取：碰到即消失，打开背包应能看到物品。
   - 按键拾取：站在范围内按 **E**。

---

## 四、Physics 2D 检查（拾取没反应时）

1. **Edit** → **Project Settings** → **Physics 2D** → **Layer Collision Matrix**  
   确认 **Player** 与 **拾取物所在 Layer** 的交叉是**勾选**的（会碰撞）。
2. 拾取物 **Rigidbody 2D**：可选；没有也行。若有，**Body Type** 用 **Kinematic** 或 **Static** 即可，避免被挤飞。
3. 确认 **Player** 的 Collider 与拾取物的 **Trigger** 能重叠（Z/排序对 2D 一般无影响，主要看 Layer）。

---

## 五、Inspector 参数说明

| 参数 | 说明 |
|------|------|
| Item | 物品数据（ScriptableObject） |
| Count | 一次拾取数量 |
| Require Interact Key | 是否需按键才拾取 |
| Interact Key | 交互键，默认 E |
| Destroy When Partial Pickup | 仅装入一部分时是否仍销毁物体（默认关：装不下会扣减 `count` 并保留物体） |
| On Picked Up | 成功拾取并销毁时触发，可接 **Audio Source** 或自定义事件 |

---

## 六、与掉落、商店的衔接

- **敌人掉落**：死亡时 `Instantiate(拾取Prefab)`，位置放在敌人脚下即可。
- **商店**：不走本脚本，直接用 `Inventory.RemoveItem` + 金币逻辑。

---

## 七、常见问题

**Q：碰到没反应？**  
检查 Player Tag、Player 是否有 Collider、Layer 矩阵、ItemData 的 **id** 是否为空。

**Q：Console 说找不到 Inventory？**  
把 **Inventory** 挂在 **Player 根物体**上，或保证带碰撞体的物体能用 `GetComponentInParent<Inventory>()` 找到。

**Q：想改成只按 F？**  
在 **World Item Pickup** 上把 **Interact Key** 改成 **F**。
